# Notebooks

Exploration interactive uniquement. Toute logique réutilisable est promue dans `src/defia/`
puis appelée depuis les notebooks (ou le Makefile) — les notebooks ne sont pas la source de
vérité du pipeline.

Convention de nommage : `NN_sujet.ipynb` (ex. `01_eda.ipynb`, `02_features_reseau.ipynb`).
